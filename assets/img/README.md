## Reemplazar una imagen

Mantén el mismo nombre y la web actualiza automáticamente.

## Recomendaciones

- **Tamaño:** 1200-1800 px lado más largo
- **Formato:** JPG calidad 80-85% (peso 150-400 KB ideal)
- **Tratamiento:** las imágenes se procesan en CSS con `contrast(1.04) saturate(.78)` para look editorial cálido. Ya vienen "desaturadas" en pantalla aunque las subas en color completo.
- **Aspecto:** cualquier proporción funciona (CSS `background-size: cover` recorta al contenedor)
