# Brand assets

Logos e iconos de marca. Reemplaza estos archivos con los tuyos
manteniendo el nombre.

## Archivos

| Archivo | Uso | Proporción / viewBox | Color |
|---|---|---|---|
| `logo.svg` | Topbar (fondo claro) | 60 × 56 | sage olive `#6f7148` |
| `logo-on-dark.svg` | Colofón (fondo oscuro) | 60 × 56 | sand `#d6c9b1` |
| `icon-sustentabilidad.svg` | Sección Esencia | 74 × 74 | ink `#16120e` |
| `icon-arquitectura.svg` | Sección Esencia | 74 × 74 | ink `#16120e` |
| `icon-futuro.svg` | Sección Esencia | 74 × 74 | ink `#16120e` |

## Recomendaciones

- Mantén el viewBox de los SVG para que las proporciones encajen
- Si reemplazas por PNG: 120 × 112 px para logo, 148 × 148 px para iconos (2x densidad)
- Los iconos deben ser monocromos para integrarse con el sistema editorial
- El logo en `logo-on-dark.svg` debe ser color claro (sand, cream, blanco) para legibilidad sobre fondo ink

## Cambios sin editar HTML

Si solo cambias el archivo SVG (mismo nombre, mismas dimensiones),
la web actualiza automáticamente al refrescar.
